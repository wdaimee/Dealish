About Dealish

Find Your Delish Deal !

Dealish is a web application for food lovers to easily find deals and happy-hours nearby. They can find restaurants through a built-in google map and easily save their favourites restaurants for later use.

Technologies Used
  • Django Framework
  • PostgreSQL ( POSTGIS )
  • Google Map API
  • HTML5
  • CSS ( Bootsrap Framework )
  • JS ( JQUERY Library)
